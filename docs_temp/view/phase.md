# view_agent 작업 Phase

## Phase 4 — 콘솔 입출력 (Controller가 넘길 데이터 형태 확정 후 시작, Phase 3과 병행 가능)

1. 메인 메뉴 출력 및 입력 프롬프트
2. 시료 등록/목록/검색 화면 (표 형태, 재고 수량 포함)
3. 주문 생성 입력 프롬프트(고객명·시료ID·수량) + 결과 출력
4. RESERVED 목록 표시 + 승인/거절 선택 UI, 처리 결과(상태 뱃지) 출력
5. 모니터링 화면
   - 상태별 주문 수 집계 (REJECTED 제외)
   - 시료별 재고 현황 (여유/부족/고갈) 표시
6. 생산 라인 현황 화면 (현재 생산 중인 시료 + FIFO 대기 큐)
7. 출고 처리 화면 (CONFIRMED 목록 + 선택/결과 출력)
